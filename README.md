## Instalation

- Clone this project
- Run Command

```
bun install
```

or

```
npm install
```

and

```
composer install
```

## Running

#### Development

```
bun run dev
```

or

```
npm run dev
```

and

```
php artisan serve
```

#### Production

```
bun run build
```

or

```
npm run build
```

and

```
Run using webserver
```
