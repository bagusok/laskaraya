<h1 align="center">Laskaraya</h1>
<p align="center">Ini adalah project pencatatan prestasi</p>

## Instalation

- Clone this project
- Run Command

```
bun install
```

or

```
npm install
```

and

```
composer install
```

## Running

#### Development

```
bun run dev
```

or

```
npm run dev
```

and

```
php artisan serve
```

#### Production

```
bun run build
```

or

```
npm run build
```

and

```
Run using webserver
```
